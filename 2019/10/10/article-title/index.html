概念

  • gRPC：基于RPC的一种远程调用协议（php平台只支持gRPC客户端调用，不支持gRPC服务端实现）
  • Protobuf：语言无关的高效数据交换协议，用于支持gRPC在不同语言间序列化和反序列化数据，具体的数据交换约定由.proto文件定义

安装protobuf

在 mac 下安装

$ brew install protobuf

在 linux 下安装 

$ tar -zxvf protobuf-all-3.5.1.tar.gz
$ cd protobuf-3.5.1
$ ./configure
$ make 
$ make install

在 Windows 下安装

参考【PHP 中使用 gRPC 客户端】

查看 protobuf 版本

$ protoc --version

安装 php 的 grpc 扩展

$ pecl install grpc

安装 php 的 protobuf 扩展

$ pecl install grpc

安装 grpc_php_plugin

在 mac 下安装

$ brew install grpc // 不包括 grpc_go_plugin 包括以下 plugin
// grpc_cli
// grpc_cpp_plugin
// grpc_csharp_plugin
// grpc_node_plugin
// grpc_objective_c_plugin
// grpc_php_plugin
// grpc_python_plugin
// grpc_ruby_plugin

其他平台可以通过以下方式安装

$ git clone -b $(curl -L https://grpc.io/release) https://github.com/grpc/grpc
$ cd grpc
$ git submodule update --init
$ make grpc_php_plugin

使用 composer 管理依赖加载

安装依赖

$ composer require "grpc/grpc" 
$ composer require "google/protobuf"

修改composer.json

"autoload": {
    ...,
    "psr-4":{
        "GPBMetadata\\":"Micro/GPBMetadata/",
        "Protobuf\\":"Micro/Protobuf/"
    }
},

$ composer dump-autoload

.proto 规范

修改 user.proto 文件 package，避免生成的 php 文件目录太深，package 对应 php 中 namespace Protobuf/User

syntax = "proto3";
// package chuman.micro.app.user.srv; 
package protobuf.user;
...

Response message 字段名为小驼峰，因为数据库中的字段名都是小驼峰。避免php需要将大驼峰转小驼峰

message UserResponse {
    int64 id=1;
    string nickname=2;
}

执行编译 .proto 命令，生成php客户端基类

// .proto 放在 gugu/Micro 下
$ cd gugu/Micro
$ protoc --php_out=../Micro --grpc_out=../Micro --plugin=protoc-gen-grpc=/usr/local/bin/grpc_php_plugin user.proto
